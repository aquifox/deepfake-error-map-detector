# Privacy Notice

이 프로젝트는 개인정보 보호를 위해 외부 추론 서버를 기본으로 사용하지 않습니다.

## 이미지 처리 방식

Chrome 확장 프로그램은 사용자가 버튼을 눌렀을 때 현재 웹페이지의 이미지 URL을 확인합니다. 분석 가능한 이미지는 브라우저 확장 프로그램이 가져온 뒤 `http://127.0.0.1:8000/predict` 로컬 서버로 전송합니다.

즉, 기본 구조에서는 이미지가 개발자 서버나 외부 API 서버로 전송되지 않습니다.

## 사용자가 알아야 할 점

- localhost 서버가 켜져 있을 때만 분석이 수행됩니다.
- 분석 대상 이미지는 사용자의 컴퓨터 내부 Python 서버로 전달됩니다.
- 민감한 이미지, 타인의 얼굴, 개인정보가 포함된 이미지는 분석 전에 주의해야 합니다.
- AI 결과는 확률적 판단이며 사실 확정이 아닙니다.

## 확장 프로그램 권한

확장 프로그램은 웹페이지 이미지에 접근하기 위해 host permission을 사용합니다. 이 권한은 페이지의 이미지를 가져와 로컬 서버로 보내기 위한 목적입니다.

공개 배포 시에는 Chrome Web Store 정책에 맞춰 권한 설명과 개인정보 처리방침을 별도로 명시해야 합니다.
